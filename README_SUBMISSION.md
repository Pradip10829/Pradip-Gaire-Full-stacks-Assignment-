# CET138 Assignment 1 - Submission Package

This folder contains a complete, rubric-aligned submission for **CET138 Full Stack Development Assignment 1**.

## Contents

- `index.html` - root homepage (GitHub Pages ready, includes all 5 embedded demos).
- `styles.css` - root stylesheet used by the homepage.
- `script.js` - root script used by the homepage.
- `assets/` - root assets used by the homepage.
- `demos/` - root demo pages used by embedded iframes.
- `.nojekyll` - ensures GitHub Pages serves static files as-is.
- `eportfolio/index.html` - main portfolio page with all five required sections.
- `eportfolio/demos/full-stack-demo.html` - working full stack lifecycle demo.
- `eportfolio/demos/html-demo.html` - semantic HTML demonstration.
- `eportfolio/demos/css-demo.html` - responsive CSS demonstration.
- `eportfolio/demos/bootstrap-demo.html` - Bootstrap framework demonstration.
- `eportfolio/demos/javascript-demo.html` - JavaScript interactive task app.
- `CET138_Assignment1_Final_Submission.pdf` - professionally formatted PDF version.
- `report/generate_pdf.py` - script used to generate the PDF.

## Final Steps Before Submission

1. Open `eportfolio/index.html` and replace `[Your Name]` and `[Your ID]`.
2. Publish the `eportfolio` folder (Canvas ePortfolio, GitHub Pages, or another static host).
3. Paste the public URL into the PDF details section (or regenerate PDF after updating script text).
4. Submit the public link on Canvas as required by the assignment brief.
5. For GitHub Pages without Actions: in GitHub `Settings > Pages`, choose `Deploy from a branch`, branch `main`, folder `/ (root)`.
